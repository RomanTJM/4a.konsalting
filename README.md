# Приложение для выбора тарифов

Веб-приложение для выбора и покупки тарифных планов

## Быстрый запуск

1. **Установка зависимостей**:
   ```bash
   npm install
   ```

2. **Запуск в режиме разработки**:
   ```bash
   npm run dev
   ```
   Откройте [http://localhost:3000](http://localhost:3000) в браузере.

3. **Сборка проекта**:
   ```bash
   npm run build
   ```

## 🛠 Технологический стек

- **Фреймворк**: Next.js 15+ (App Router)
- **Библиотека**: React 19
- **Стилизация**: Tailwind CSS v4 & Native CSS Variables
- **Язык**: TypeScript

## 🌟 Ключевые особенности

- **Динамические данные**: Интеграция с внешним API для получения актуальных цен и описаний тарифов.
- **Умный таймер**: Обратный отсчет (2 минуты) с визуальными эффектами: при остатке менее 30 секунд таймер начинает мигать красным. По истечении времени скидки скрываются с плавной анимацией.
- **Премиальный UI/UX**:
  - Адаптивная верстка (Desktop/Mobile).
  - Плавные цветовые переходы и микро-анимации.
  - Кастомное модальное окно оплаты с анимацией успеха.
- **Оптимизация активов**: Локальное хранение изображений для ускорения загрузки и независимости от внешних ресурсов.
- **Валидация**: Реализована проверка принятия условий оферты перед совершением покупки.

## 📁 Структура проекта

- `/app` — Основная логика страниц и глобальные стили.
- `/components` — Переиспользуемые React-компоненты (Header, TariffCard, Modal и др.).
- `/public` — Статические ассеты (изображения, иконки).
- `/types` — Описания типов TypeScript.
- `/api` — Прокси-роуты для работы с внешними данными.
